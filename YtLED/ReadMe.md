**瑞合信屏卡**

型号: T8<br>
配置:

1. 添加区域 modbus从站<br>
2. 设置参数

* 波特率: 键大 9600,京创 4800<br>
* 从站地址: 5<br>
* 数据地址: 1

**建大传感器**

型号: rs-ws-n01-1a<br>
配置: set键进入

* 1.设备地址
* 2.波特率
* 数据地址 1

**配置文件**

| 名称       | 例子   | 描述                 |
|----------|------|--------------------|
| port     | com1 | 485端口              |
| baud     | 9600 | 波特率                |
| databits | 8    | 数据位                |
| stopbits | 1    | 启停位                |
| parity   | none | 校验位(none,odd,even) |
| card     | 5    | 屏卡485地址            |
| cdAddr   | 1    | 屏卡数据地址             |
| cdEndian | 2    | 大小端(1,大端;2,小端)     |
| slave    | 1    | 传感器485地址           |
| address  | 1    | 温度数据地址             |
| endian   | 1    | 大小端(1,大端;2,小端)     |
| quantity | 1    | 数据长度(未用)           |
| expire   | 0    | 过期时间(未用)           |

**重点**

1、京创传感器 endian=1,使用大端模式。
