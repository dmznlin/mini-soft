**瑞合信屏卡**

型号: T8<br>
配置:

1. 添加区域 modbus从站<br>
2. 设置参数

* 波特率: 键大 9600,京创 4800<br>
* 从站地址: 5<br>
* 数据地址: 1

**建大传感器**

型号: rs-ws-n01-1a<br>
配置: set键进入

* 1.设备地址
* 2.波特率
* 数据地址 1

**配置文件**

| 名称       | 例子   | 描述                 |
|----------|------|--------------------|
| port     | com1 | 485端口              |
| baud     | 9600 | 波特率                |
| databits | 8    | 数据位                |
| stopbits | 1    | 启停位                |
| parity   | none | 校验位(none,odd,even) |
| card     | 5    | 屏卡485地址            |
| cdAddr   | 1    | 屏卡数据地址             |
| cdEndian | 2    | 大小端(1,大端;2,小端)     |
| slave    | 1    | 传感器485地址           |
| address  | 1    | 温度数据地址             |
| endian   | 1    | 大小端(1,大端;2,小端)     |
| quantity | 1    | 数据长度(未用)           |
| expire   | 0    | 过期时间(未用)           |

**重点**

1、京创传感器 endian=1,使用大端模式。

**合宙刷固件**

1、安装 [Luatools](https://docs.openluat.com/common/Luatools/) <br>
2、点击下载固件,选择 `固件路径` 后点击下载

**合宙模块配置**

1、启动 mqtt 客户端，`服务器: broker.hivemq.com:1883  主题: yt202603/pub/+`<br>
2、插入手机卡，启动模块。模块启动成功后，会发送上线通知。通知中有模块的id，如：`yt202603/sub/862289057685347`<br>
4、向该主题发送 `{"cmd":2,"cfg":"yt"}`，获取当前的配置信息。<br>
5、修改配置信息后，发送上传。成功会返回 `{"data":"/yt.json","cmd":3,"cfg":"yt"}`
